# Campus Lost and Found System

## Project Description
The Campus Lost and Found System is a web-based application designed to help students and staff report, search, and recover lost items on campus. Users can submit details of lost or found items, browse available listings, and contact the item owner.

This system improves communication between students and increases the chances of recovering lost belongings.

---

## Features

- Add lost or found items
- View all reported items
- Search and filter items
- Mark items as resolved
- Separate pages for Lost and Found items
- Secure backend with input validation
- Database storage using MySQL
